# CodeLab

Run Code Servers on your own personal AWS infrastructure.

## Prerequisites

- [AWS CloudShell](https://console.aws.amazon.com/cloudshell/home)

## Installation

   ```sh
   npm install https://github.com/primordial-stew/codelab
   ```

## Usage

   ```sh
   npx codelab up <repo>
   ```

   ```sh
   npx codelab down
   ```
